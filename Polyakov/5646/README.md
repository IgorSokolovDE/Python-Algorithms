# Задача № 5646

Раздел № 156: Поиск заданных последовательностей символов

Определите максимальное число в этом файле, ограниченное двумя парами символов KK и удовлетворяющее маске «43??78???34», где символ ? обозначает любую цифру. Пример такого числа: 43127812334. Найдите произведение нечётных цифр найденного числа.

Ответ:
229635
