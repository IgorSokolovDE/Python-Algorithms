# Задача № 5959
#### Задание КИМ № 6: Анализ программ с циклами

Раздел № 175: Циклические программы для исполнителей

(Б. Михлин) Исполнитель Черепаха действует на плоскости с декартовой системой координат. В начальный момент Черепаха находится в начале координат, её голова направлена вдоль положительного направления оси ординат, хвост опущен. При опущенном хвосте Черепаха оставляет на поле след в виде линии. В каждый конкретный момент известно положение исполнителя и направление его движения. . У исполнителя существует 6 команд: Поднять хвост, означающая переход к перемещению без рисования; Опустить хвост, означающая переход в режим рисования; Вперёд n (где n – целое число), вызывающая передвижение Черепахи на n единиц в том направлении, куда указывает её голова; Назад n (где n – целое число), вызывающая передвижение в противоположном голове направлении; Направо m (где m – целое число), вызывающая изменение направления движения на m градусов по часовой стрелке, Налево m (где m – целое число), вызывающая изменение направления движения на m градусов против часовой стрелки.
Запись
Повтори k [Команда1 Команда2 … КомандаS]
означает, что последовательность из S команд повторится k раз. Черепахе был дан для исполнения следующий алгоритм:
Повтори k [ 
  Повтори 4 [ Вперед 30 Направо 90 ] Поднять хвост
  Вперед 5 Направо 90 Вперед  5 Опустить хвост 
  Направо 270 ]
При каком значении параметра k нарисованная на поле траектория Черепахи пройдет несколько раз ровно через 20 точек?

Ответ:
5
