# Задача № 5727
#### Задание КИМ № 12: Алгоритмы для исполнителей с циклами и ветвлениями
Раздел № 57: Исполнитель Редактор

(А. Рогов) Исполнитель Редактор получает на вход строку цифр и преобразовывает её. Редактор может выполнять две команды, в обеих командах v и w обозначают цепочки символов.
1. заменить (v, w) 
2. нашлось (v)

Первая команда заменяет в строке первое слева вхождение цепочки v на цепочку w. Если цепочки v в строке нет, эта команда не изменяет строку. Вторая команда проверяет, встречается ли цепочка v в строке исполнителя Редактор.
Дана программа для исполнителя Редактор:
НАЧАЛО
__ПОКА нашлось(21) ИЛИ нашлось(31) ИЛИ нашлось(32)
____ЕСЛИ нашлось(21) 
______ТО заменить(21, 12)
____КОНЕЦ ЕСЛИ
____ЕСЛИ нашлось(31) 
______ТО заменить(31, 13)
____КОНЕЦ ЕСЛИ
____ЕСЛИ нашлось(32) 
______ТО заменить(32, 23)
____КОНЕЦ ЕСЛИ
__КОНЕЦ ПОКА
КОНЕЦ
На вход программы поступает строка из n цифр, содержащая равное количество цифр 1, 2 и 3, расположенных в произвольном порядке. При каком минимальном значении n в строке, полученной в результате работы программы, в позиции 50 будет стоять цифра 2? Цифры в строке нумеруются последовательно слева направо, начиная с 1.

Ответ:
75
