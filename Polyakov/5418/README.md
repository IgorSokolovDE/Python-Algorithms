# Задача № 5418
#### Задание КИМ № 16: Вычисление значения рекурсивной функции

Раздел № 44: Вычисление значения рекурсивной функции

(А. Богданов) Алгоритм вычисления значения функции F(n), где n – целое неотрицательное число, задан следующими соотношениями:
F(n) = n, если n < 2
F(n) = F(n / 2) + 1, если n ? 2 и число n чётное,
F(n) = F(3n + 1) + 1, если n ? 2 и число n нечётное.
Определите количество значений n на отрезке [1;100000], для которых F(n) равно 16.

Ответ:
24
