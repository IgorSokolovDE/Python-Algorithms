# Задача № 5492
#### Задание КИМ № 14: Позиционные системы счисления

Раздел № 174: Неизвестные цифры в записи чисел в разных системах

(А. Богданов) Операнды арифметического выражения записаны в системе счисления с основанием 13.
8x121(??) – 7x575(??)

В записи чисел переменной x обозначена неизвестная цифра из алфавита 13-ричной системы счисления. Определите наименьшее значение x, при котором значение данного арифметического выражения кратно 19. Для найденного значения x вычислите частное от деления значения арифметического выражения на 19 и укажите его в ответе в десятичной системе счисления. Основание системы счисления в ответе указывать не нужно.

Ответ:
1464
