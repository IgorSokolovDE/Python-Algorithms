# Задача № 5018

### Раздел № 170: Анализ чисел из файла (2+ прохода)

Определите сначала количество пар, в которых оба элемента больше, чем сумма всех цифр «4» в пятеричной записи всех чисел в файле, кратных 12, а затем максимальную из сумм таких пар. Под парой подразумевается два идущих подряд элемента последовательности.

Ответ: 
93 1951
