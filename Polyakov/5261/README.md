# Задача № 5261
### Задание КИМ № 24: Обработка символьных строк
### Раздел № 155: Цепочки максимальной длины

(Е. Джобс) Текстовый файл 24-209.txt содержит строку из набора A, B, C, D, E, F, всего не более чем из 106 символов. Найдите максимальное количество подряд идущих пар символов АB, CB, BС и BA, стоящих одна за другой и пересекающихся с соседними парами одной буквой. Например, в строке BDEABCBABBD такие пары составляют подстроку ABCBAB = AB + BC + CB + BA + AB, итого 5 пар.

Ответ:
16
