# Задача № 5742
#### Задание КИМ № 16: Вычисление значения рекурсивной функции
Раздел № 44: Вычисление значения рекурсивной функции

(М. Байрамгулов) Алгоритм вычисления функции F(n, m), где n и m – натуральные числа, задан следующими соотношениями:
F(n, m) = 0, если m > n,
F(n, m) = 1 + F(n, m + 1), если m ? n и n делится на m,
F(n, m) = F(n, m + 1), если m ? n и n не делится на m.
Чему равно значение выражения F(107864, 3)?

Ответ:
14
