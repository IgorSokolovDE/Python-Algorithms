# Задача № 5226
#### Задание КИМ № 25: Обработка целочисленных данных. Поиск делителей
Раздел № 157: Перебор чисел. Определение количества делителей числа

Пусть N(k) = 19 500 000 + k, где k – натуральное число. Найдите пять наименьших значений k, при которых N(k) нельзя представить в виде произведения трёх натуральных чисел, больших 1. В ответе запишите найденные значения k в порядке убывания, справа от каждого значения запишите наибольший делитель N(k), не равный самому числу.

Ответ:
11 62701
9 6500003
7 201031
2 9750001
1 182243
