# Задача № 5328
#### Задание КИМ № 17: Обработка целочисленных данных. Проверка делимости
Раздел № 170: Анализ чисел из файла (2+ прохода)

(ЕГЭ-2022) В файле 17-338.txt содержится последовательность целых чисел. Элементы последовательности могут принимать целые значения от 1 до 100 000 включительно. Определите количество пар элементов последовательности, в которых остаток от деления хотя бы одного из элементов на 117 равен минимальному элементу последовательности. В ответе запишите количество найденных пар, затем максимальную из сумм элементов таких пар. В данной задаче под парой подразумевается два идущих подряд элемента последовательности.
