# Задача № 5731
#### Задание КИМ № 16: Вычисление значения рекурсивной функции
Раздел № 44: Вычисление значения рекурсивной функции

(А. Кабанов) Алгоритм вычисления значения функции F(n), где n – натуральное число, задан следующими соотношениями:
F(n) = n, если n ? 10 000,
F(n) = n/4 + F(n/4 + 2), если n < 10 000 и n делится на 4,
F(n) = 1 + F(n + 2) , если n < 10 000 и n не делится на 4.
Чему равно значение выражения F(174) – F(3)?

Ответ:
67
