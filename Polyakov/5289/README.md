# Задача № 5289
#### Задание КИМ № 25: Обработка целочисленных данных. Поиск делителей
Раздел № 159: Перебор чисел. Числа с заданными свойствами

(М. Фирсов) Пусть D(N) – наибольший делитель числа N, отличный от самого числа, и Q(N) – записанная в обратном порядке сумма всех его простых сомножителей (необязательно различных). Найдите первые 5 натуральных чисел N, для которых N + D(N) + Q(N) > 202122. В ответе запишите все найденные числа в порядке возрастания, справа от каждого запишите сумму D(N) + Q(N) для этого N.

Ответ:
68594 133540
69374 133330
69754 132720
69794 134740
71194 135150
