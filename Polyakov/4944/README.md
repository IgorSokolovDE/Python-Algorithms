# Задача № 4944
#### Задание КИМ № 25: Обработка целочисленных данных. Поиск делителей
Раздел № 159: Перебор чисел. Числа с заданными свойствами

Особыми будем называть нетривиальные делители числа, сумма цифр которых равна 20. Нетривиальными считаются все делители, кроме 1 и самого числа. Пусть D(N) – шестой по величине (считая с наибольшего) особый делитель натурального числа N. Если у числа N меньше 6 различных особых делителей, то принимаем D(N) = 0. Найдите 5 наименьших натуральных чисел, превышающих 400 000 000, для которых D(N) > 0. В ответе запишите для каждого найденного N сначала значение D(N), а затем общее количество особых делителей (в порядке возрастания соответствующих чисел N).

Ответ:

965 6
2756 6
14960 9
5285 9
9380 7
