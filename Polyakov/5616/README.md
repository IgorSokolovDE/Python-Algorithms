# Задача № 5616
#### Задание КИМ № 25: Обработка целочисленных данных. Поиск делителей
Раздел № 172: Перебор чисел. Соответствие маске

(К. Багдасарян) Назовём маской числа последовательность цифр, в которой также могут встречаться символ «Р», означающий произвольное простое число. Например, маске 1P2 соответствуют числа 1132, 10132, 10032 и т.д. Среди натуральных чисел, не превышающих 107, найдите все числа, соответствующие маске 3P1, делящиеся на 9797 без остатка. В ответе запишите в первом столбце таблицы все найденные числа в порядке возрастания, а во втором столбце – соответствующие им результаты деления этих чисел на 9797.

Ответ:
3360371 343
3556311 363
3948191 403
