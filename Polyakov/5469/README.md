# Задача № 5469
#### Задание КИМ № 16: Вычисление значения рекурсивной функции
Раздел № 44: Вычисление значения рекурсивной функции

(Е. Джобс) Алгоритм вычисления значения функции F(n), где n – целое неотрицательное число, задан следующими соотношениями:
F(n) = 3·n, если n < 3
F(n) = F(n – 2)·F(n – 1) - n, если n > 2 и число n чётное,
F(n) = F(n – 1) - F(n - 2) + 2·n, если n > 2 и число n нечётное.
Определите последние две цифры значения F(30).

Ответ:
36
