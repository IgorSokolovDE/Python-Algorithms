# Задача № 5335
#### Задание КИМ № 8: Комбинаторика
Раздел № 145: Сколько чисел можно составить при заданных ограничениях?

(ЕГЭ-2022) Определите количество пятизначных чисел, записанных в восьмеричной системе счисления, в записи которых ровно одна цифра 6, при этом никакая нечётная цифра не стоит рядом с цифрой 6.

Ответ:
2961
