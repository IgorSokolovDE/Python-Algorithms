# Задача № 5283
#### Задание КИМ № 17: Обработка целочисленных данных. Проверка делимости
Раздел № 170: Анализ чисел из файла (2+ прохода)

(И. Кушнир) В файле 17-335.txt содержится последовательность целых чисел. Элементы последовательности могут принимать целые значения от 1000 до 9999 включительно. Обозначим через M минимальное число в последовательности, кратное 43. Определите количество пар последовательности, в которых либо сумма чисел кратна M, либо хотя бы в одном из чисел последняя цифра совпадает с последней цифрой числа M. Гарантируется, что такая пара в последовательности есть. В ответе запишите количество найденных пар, затем максимальное из чисел, которые являются элементами таких пар. В данной задаче под парой подразумевается два идущих подряд элемента последовательности.

Ответ:

1895 9984
