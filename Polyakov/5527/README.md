# Задача № 5527
### Задание КИМ № 17: Обработка целочисленных данных. Проверка делимости
### Раздел № 170: Анализ чисел из файла (2+ прохода)

(М. Ишимов) В файле 17-345.txt содержится последовательность целых чисел. Элементы последовательности могут принимать целые — значения от 1 до 10 000 включительно. Определите количество пар последовательности, в которых только одно число меньше разности максимального и минимального из чисел последовательности, оканчивающихся на 52.
В ответе запишите количество найденных пар, затем максимальную из сумм элементов таких пар. В данной задаче под парой подразумевается два идущих подряд элемента последовательности.
