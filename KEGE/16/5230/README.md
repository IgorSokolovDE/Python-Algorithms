#### № 5230 (Уровень: Сложный)
Алгоритм вычисления значения функции F(n, k), где n, k – натуральные числа, задан следующими соотношениями: 

F(n, k) = 0 при k = 0; 
F(n, k) = F(n, k - 1) + k^2 при k > 0 и n mod k = 0;
F(n, k) = F(n, k - 1) при k > 0 и n mod k ? 0.

Чему равно значение функции F(11223456789, 123456789)?

Примечание: mod возвращает остаток от деления одного числа на другое нацело.

Ответ:
7569616146557181
